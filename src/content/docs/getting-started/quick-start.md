## 准备条件

开始前先确认以下信息：

- 一台可以持续运行的设备
- 已整理好的媒体文件目录
- 可以访问 Emby Web 管理界面的浏览器

## 基础流程

1. 安装并启动 Emby Server。
2. 打开 Web 管理页面，完成管理员账号初始化。
3. 创建第一个媒体库，例如「电影」或「电视剧」。
4. 指定媒体目录，等待 Emby 扫描并抓取元数据。
5. 在电视、手机或浏览器端登录并验证播放是否正常。

## 首次配置建议

- 先把目录结构整理好，再导入媒体库，后续维护成本会低很多。
- 管理员密码不要留空，尤其是计划做远程访问时。
- 如果媒体很多，先导入一小部分做命名与刮削测试，再批量加入。

## 完成标志

当你能在客户端看到海报墙、进入详情页并成功播放一部影片时，说明基础搭建已经完成。
