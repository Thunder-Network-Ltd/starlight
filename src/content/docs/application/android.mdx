## Yamby

一款第三方 Emby 客户端，基于 Material Design 3 设计，轻量、强大，或许是目前速度最快的客户端 (源自作者简介)；同时支持手机和平板设备。

### 下载安装

前往 [Google Play](https://play.google.com/store/apps/details?id=com.hush.yamby&pli=1) 或 [官方 Telegram 频道](https://t.me/yamby_release) 下载。

## Hills

支持 mpv 和 ExoPlayer 双播放内核，可自由切换，完美适配不同设备。
支持硬解杜比视界、HDR10+、4K，轻松解码高清内容，呈现震撼视觉效果
支持特效字幕、双字幕，可自定义字体、颜色、大小、位置等，打造个性化观影体验
支持聚合搜索、聚合播放源、聚合视界，一站式获取丰富片源，看片更高效
弹幕、Anime4K超分辨率、自定义mpv配置、跳过片头等其他丰富功能

### 下载安装
前往 [Google Play](https://play.google.com/store/apps/details?id=com.mountains.hills) 或 [官方 Telegram 频道](https://t.me/Hills_app) 下载。
